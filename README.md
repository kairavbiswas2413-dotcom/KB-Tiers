KB Tiers
========

Client companion mod to assign player tiers (GUI + keybind) and show prefixes locally or server-side if the server also runs the mod.

Build: GitHub Actions builds jar automatically. Download artifact from Actions -> build -> Artifacts.

Install:
- Put the built jar into your Minecraft client's mods/ folder to use GUI and view tiers locally.
- (Optional) Put jar on a server to allow server-side persistence and to make tier prefixes visible to everyone.

Defaults:
- Verifier username: KB_PlazZ
- Verifier code: 331025
- Config & tiers saved to config/cctl_tiertagger/
